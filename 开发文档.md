### 开发文档

安装gulp:

	npm install gulp -g
	npm install webpack -g
	npm install  webpack-cli -g

测试是否安装成功:

	
	gulp -v
	webpack -v
	
---上面无效

安装:

	npm install

运行:


	npm run dev


打包:

	npm run dist


上传打包文件:

	修改package.json的版本号：

	  "version": "1.0.2",


打包上传:

	登陆:

	npm adduser 

	打包：

	npm publish
